= ホストポートプロトコルを変更します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ホストのプロトコルを別のプロトコルに変換して互換性を確保したり、ホスト接続のプロトコルをインフラのニーズに合わせて別のプロトコルに変換したりできます。
