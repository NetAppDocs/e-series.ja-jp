= ホストでストレージアクセスを確認
:icons: font
:imagesdir: ../media/


[role="lead"]
ボリュームを使用する前に、ホストがボリュームに対してデータの読み取りと書き込みを実行できることを確認します。

ファイルシステムでフォーマットされた初期化済みボリューム。

.手順
. ホストで、 1 つ以上のファイルをディスクのマウントポイントにコピーします。
. ファイルを元のディスク上の別のフォルダにコピーします。
. 「 IFF 」コマンドを実行して、コピーしたファイルを元のファイルと比較します。


コピーしたファイルとフォルダを削除します。
