両方のコントローラのI/OモジュールまたはI/Oモジュールブランクを交換します。

.このタスクについて
まず一方のコントローラーから始め、次に他方のコントローラーについても同様の手順を繰り返します。

.手順
. まだ接地していない場合は、自分で適切に接地してください。
. いずれかのコントローラで、ターゲットスロットからI/OモジュールまたはI/Oブランキングモジュールを取り外します。
+
次の図は、I/Oブランキングモジュールを取り外しているところを示しています。

+
image::../media/drw_g_io_blanking_module_replace_ieops-1901.svg[I/Oブランキングモジュールを取り外します]

+
[cols="1,4"]
|===


 a| 
image:../media/icon_round_1.png["コールアウト番号1"]
 a| 
I/OモジュールまたはI/Oブランキングモジュールで、つまみネジを反時計回りに回して緩めます。



 a| 
image:../media/icon_round_2.png["コールアウト番号2"]
 a| 
左側のタブとつまみネジを使用して、I/OモジュールまたはI/Oブランキングモジュールをコントローラから引き出します。

|===
. 新しいI/Oモジュールを取り付けます：
+
.. I/Oモジュールをコントローラスロット開口部の端に合わせます。
.. I/Oモジュールをスロットにゆっくりと奥まで押し込み、モジュールがコネクタに正しく装着されていることを確認してください。
+
左側のタブとサムスクリューを使ってI/Oモジュールを押し込むことができます。

.. つまみネジを時計回りに回して締めます。


. I/Oモジュールをホストネットワークにケーブルで接続します。
+
link:../install-hw-ef50-ef80/install-cable.html["ハードウェアをケーブルで接続する"]セクションは、システムのインストールとセットアップに関するコンテンツのホストネットワークケーブル配線に使用できます。

. これらの手順を繰り返して、もう一方のコントローラにI/Oモジュールを追加します。
