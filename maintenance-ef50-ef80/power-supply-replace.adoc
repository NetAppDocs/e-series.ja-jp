= 電源装置の交換 - EF50およびEF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
EF50またはEF80ストレージ システムの電源装置ユニット（PSU）に障害が発生した場合や故障した場合は交換し、システムが安定した動作に必要な電力を継続的に受け取れるようにします。

.作業を開始する前に
* Recovery Guruで詳細を確認し、電源に問題があることを確認してください。
* 電源ユニットの注意（オレンジ色）LEDが点灯していることを確認します。これは、電源ユニットまたは内蔵ファンに障害があることを示しています。
* その他のストレージシステムの問題に対処します。
+
Recovery Guruで*再確認*を選択して、他に問題がないか確認してください。

* 次のものがあることを確認します。
+
** 交換用電源装置
+

NOTE: 異なる電圧タイプの PSU を混在させないでください。いつものように同じように置換します。

** ESDリストバンドを着用しているか、その他の静電気対策を講じている。
** 平らで静電気のない作業面。
** 障害が発生したコントローラのSANtricity System Managerにアクセスできるブラウザを備えた管理ステーション。
+
System Manager インターフェイスを開くには、ブラウザで障害が発生したコントローラのドメイン名または IP アドレスにアクセスしてください。





.このタスクについて
* 電源装置に障害が発生した場合は、ストレージ システムに冗長電源を確保するために、できるだけ早く交換する必要があります。
* 電源を交換するには、故障した PSU から電源コードを抜き、交換してから電源コードを再接続します。
* ストレージシステムの場所（青色）のLED（ストレージシステム前面および両方のコントローラ上）を点灯させることで、該当するストレージシステムを物理的に特定するのに役立ちます。SANtricity System Managerを使用し、*ハードウェア* ＞ *コントローラとコンポーネント* を選択し、*コントローラシェルフ* タブを選択して、コンテキストメニューから *ロケーターライトを点灯* を選択します。


.手順
. まだ接地していない場合は、自分で適切に接地してください。
. 電源コードリテーナーを開いてPSUから電源コードを外し、PSUから電源コードのプラグを抜きます。
+

NOTE: 電源ユニットには電源スイッチがありません。

. PSUを取り外します：
+
image::../media/ef50-ef80_psu_ac_orange_tab_replace_ieops-2611.svg[AC PSUを交換する]

+
[cols="1,4"]
|===


 a| 
image::../media/icon_round_1.png[コールアウト番号1]
 a| 
電源ユニットのハンドルを上方向に回転させて水平位置にし、それからハンドルを握ってください。



 a| 
image::../media/icon_round_2.png[コールアウト番号2]
 a| 
親指でオレンジ色のロックタブを押して、PSUをコントローラーから取り外します。



 a| 
image::../media/icon_round_3.png[コールアウト番号3]
 a| 
もう一方の手で重さを支えながら、コントローラから PSU を引き抜きます。


CAUTION: PSUは短いです。コントローラから取り外す際は、必ず両手で支えてください。そうしないと、突然コントローラから外れて怪我をする可能性があります。

|===
. 交換用電源装置を開梱し、ストレージシステムの近くの平らな場所に置いてください。
+
梱包材は、障害が発生した電源装置を返送するときのためにすべて保管しておいてください。

. 交換用PSUを取り付けます：
+
.. 両手を使って、PSUの端をコントローラーの開口部に合わせて支え、位置を合わせます。
.. オレンジ色のロックタブがカチッと音がするまで、PSUをコントローラーにゆっくりと押し込んでください。
+
電源ユニットは、内部コネクタに正しく接続され、所定の位置に固定されるのは一方向のみです。

+

NOTE: 内部コネクタの損傷を防ぐため、PSUをコントローラに差し込む際に過度の力を加えないでください。

.. ハンドルを下に回転させて、通常の操作の邪魔にならないようにします。


. 電源コードをPSUに再接続し、電源コードリテーナーで電源コードを固定します。
+
PSUへの電力供給が復旧すると、ステータスLEDが緑色に点灯します。

. PSUのステータスが「最適」であり、ストレージシステムの注意喚起LEDが消灯していることを確認してください。
+
ステータスが最適でない場合、または注意LEDのいずれかが点灯している場合は、電源コードが正しく接続されていること、および電源ユニットが正しく取り付けられていることを確認してください。必要に応じて、電源ユニットを取り外して再度取り付けます。

+

NOTE: 問題が解決しない場合は、この手順を続行する前に https://mysupport.netapp.com/site/global/dashboard["ネットアップサポート"]にお問い合わせください。

. SANtricity System Managerで、*サポート > アップグレードセンター* をクリックして、最新バージョンのSANtricity OSがインストールされていることを確認してください。
+
必要に応じて、最新バージョンをインストールします。

. 障害のある部品は、キットに付属する RMA 指示書に従ってネットアップに返却してください。
+
RMA番号が必要な場合、または交換手順に関する追加のサポートが必要な場合は、 https://mysupport.netapp.com/site/global/dashboard["ネットアップサポート"]888-463-8277（北米）、00-800-44-638277（ヨーロッパ）、または+800-800-80-800（アジア太平洋）のテクニカルサポートまでお問い合わせください。//2025-11-17 ontap-systems-internal/issues/1391
