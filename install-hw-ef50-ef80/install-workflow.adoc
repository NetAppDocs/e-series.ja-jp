= インストールとセットアップ - EF50 および EF80
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
EF50またはEF80ストレージシステムを設置およびセットアップするには、設置場所の準備、ハードウェアコンポーネントの設置と配線、ストレージシステムの電源投入、およびストレージシステムのセットアップと構成の完了が必要です。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1つ"]link:install-prepare.html["設置を準備"]
[role="quick-margin-para"]
設置準備として、設置場所を整え、ストレージシステムの吊り上げに関する注意事項を理解し、箱を開梱して内容物を梱包明細書と照合し、サポート特典を受けるためにストレージシステムを登録してください。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["2 つ"]link:install-hardware.html["ハードウェアを設置する"]
[role="quick-margin-para"]
ハードウェアを設置するには、ラックマウント用ハードウェアをキャビネットまたはラックに取り付け、次にストレージシステムを設置して固定します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["3つ"]link:install-cable.html["ハードウェアをケーブルで接続する"]
[role="quick-margin-para"]
ハードウェアをケーブル接続するには、まずコントローラ同士をケーブルで接続してコントローラ間ミラーリングを有効にします。これにより、システム全体の冗長性が確保され、キャッシュミラーリングやI/O転送に使用されます。次に、コントローラをホストネットワークにケーブルで接続します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["4."]link:install-power-hardware.html["ストレージシステムの電源をオンにします"]
[role="quick-margin-para"]
ストレージシステムの電源を入れるには、各コントローラ電源に電源ケーブルを接続し、それらを異なる回路の電源に接続します

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["5 つ"]link:install-complete-setup.html["ストレージシステムのセットアップを完了する"]
[role="quick-margin-para"]
ストレージシステムのセットアップを完了するには、コントローラの管理ポートにケーブルを接続して設定し、ストレージシステムを構成および管理してから、ホスト I/O モジュールのプロトコルが正しいことを確認してください。
