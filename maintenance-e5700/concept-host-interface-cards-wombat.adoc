= ホストインターフェイスカード
:icons: font
:imagesdir: ../media/


[role="lead"]
ホストインターフェイスカード（ HIC ）は、コントローラキャニスターにオプションで取り付けることができます。E5700 コントローラには、コントローラカード自体に搭載されたホストポートとオプションの HIC に搭載されたホストポートがあります。コントローラに搭載されたホストポートのことをベースボードホストポートと呼び、HIC に搭載されたホストポートのことを HIC ポートと呼びます。
